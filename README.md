📊 Netflix Data Analysis & Interactive Dashboard

This project analyzes the Netflix titles dataset to understand trends in content distribution, genres, ratings, growth over time, and country-wise production. Using Python data analysis and visualization libraries, an interactive dashboard was created to explore key insights about Netflix content.

🚀 Project Overview

The goal of this project is to analyze Netflix content and identify patterns such as:

Growth of Netflix content over the years

Most common genres available on Netflix

Distribution of content ratings

Countries contributing the most content

Audience type targeted by Netflix

An interactive dashboard allows users to explore these insights dynamically.

📂 Dataset Features

The dataset contains information about Netflix movies and TV shows with the following attributes:

show_id – Unique identifier for each title

type – Movie or TV Show

title – Title name

director – Director of the content

country – Country where the content was produced

date_added – Date the title was added to Netflix

release_year – Year the content was originally released

rating – Age classification of the content

duration – Duration of the movie or number of seasons

listed_in – Genre categories

year_added – Extracted year from date_added

month_added – Extracted month from date_added

📈 Dashboard Visualizations

The dashboard includes the following charts:

🎬 Genre Distribution

Shows the most common genres available on Netflix.

⭐ Rating Distribution

Displays the distribution of titles by audience rating categories.

📈 Content Growth Over Time

Shows how the number of Netflix titles has increased over the years.

🌍 Country Contribution

Displays which countries produce the most Netflix content.

These charts are interconnected through filters, allowing users to analyze the dataset dynamically.

🛠️ Technologies Used

Python

Pandas – Data cleaning and manipulation

NumPy – Numerical computations

Matplotlib & Seaborn – Data visualization

Plotly – Interactive charts and dashboard

WordCloud – Text visualization

Jupyter Notebook – Development environment

📊 Key Insights

Netflix content has grown rapidly over recent years.

Genres like Drama, Comedy, and International Movies dominate the platform.

Most content is rated TV-MA and TV-14, indicating focus on adult and teen audiences.

The United States contributes the largest share of Netflix content production.
